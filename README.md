-- Overview -- 

My dice roll app is a simple android application built using Kotlin. When the user presses a button then two dice spin simultaneously and the UI will update dispalying corresponding dice images and their numerical value.

-- Features --

Dual Dice Roll: Rolls two dice simultaneously with a single button press.
Dynamic Images: Updates dice faces based on the roll result.
Enhamcent: Text Output displaying the numeric value of each dice roll.

-- Setup --

Prerequisites
Android Studio (Arctic Fox or newer)
Kotlin support enabled
Minimum SDK: 21 (Android 5.0 Lollipop)

Steps to Run
1. Clone the repository.
2. Open the project in Android Studio.
3. Sync Gradle files to ensure all dependencies are installed.
4. Connect a physical or virtual Android device.
5. Run the application.
